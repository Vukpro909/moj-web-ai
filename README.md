# Google UI Python Chat 🤖

A modern, clean, and open-source cross-platform chat application built with Python and Kivy. This app utilizes the super-fast **Groq API** (running the `llama3-8b-8192` model) to deliver near-instant responses with a sleek Google-inspired dark mode UI.

---

## ✨ Features

* **Sleek Dark UI:** Google-inspired dark mode layout designed for mobile and desktop screens.
* **Asynchronous Requests:** Powered by Python's `threading` to keep the UI smooth and responsive while fetching AI responses.
* **Open Source & Secure:** No hardcoded API keys! Your Groq API key is saved locally on your device in a secure text file (`groq_key.txt`), making this repository 100% safe to fork and share.
* **Persistent Chat Context:** Remembers the conversation history during the session for smart, contextual replies.

---

## 🚀 How to Run

### On Mobile (Android / iOS)
1. Download **Pydroid 3** (Android) or **Pyto** (iOS) from your app store.
2. Clone or download this repository and copy the code from `main.py`.
3. Paste the code into your app, press the **Play** button, enter your Groq API Key at the top, and click **Save**.
